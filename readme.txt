== Description ==

[Cresta Addons for Elementor](https://crestaproject.com/downloads/cresta-addons/) adds many useful Elementor widgets to your website.
These widgets will allow you to add new elements to your web pages quickly and easily with a style never seen before.

### [Elementor](https://wordpress.org/plugins/elementor/) page builder is required for this plugin.

== Installation ==

1. Unzip cresta-addons-for-elementor.zip
2. Upload the folder 'cresta-addons-for-elementor' to the '/wp-content/plugins/' directory
3. Activate the plugin through the 'Plugins' menu in WordPress Dashboard
4. Go to your WordPress Dashboard under "Main Menu -> Cresta Addons" to set the options
